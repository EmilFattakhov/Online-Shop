# Online-Shop
Destination for all shopping lovers to buy and sell their products for cozy home. Built using React and Ruby on Rails

#### Application is built on Ruby on Rails RESTful API, ReactJS, PostgreSQL
#### App has Authentication, Authorization features, has full CRUD functionality
